# Spam Email Classifier (ML Project)

## Overview
This project is a simple machine learning model that classifies emails as Spam or Not Spam.

## Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy

## Features
- Text preprocessing using TF-IDF
- Naive Bayes classification
- Accuracy evaluation

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python spam_classifier.py

## Output
- Predicts whether an email is spam or not
- Displays accuracy of the model
